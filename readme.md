# Customer Behaviour Analysis Dashboard (Power BI)

## Project Overview

This project analyzes customer purchasing behavior using Power BI.
The dashboard provides insights into sales trends, customer demographics, seasonal demand, and payment preferences.

## Dataset

The dataset used in this project contains information about:

* Customer demographics
* Product categories
* Purchase amount
* Payment methods
* Seasonal purchasing trends

## Dashboard Pages

### 1. Sales Overview

This page includes:

* Total Revenue
* Total Customers
* Average Purchase
* Sales by Category
* Gender Distribution
* Top Purchased Items

### 2. Customer Behaviour Insights

This page includes:

* Sales by Season
* Purchase Frequency Analysis
* Payment Method Distribution
* Interactive Filters (Gender, Category, Season, Payment Method)

* Power BI
